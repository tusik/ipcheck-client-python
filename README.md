# Global Ping project
It's the client program for [https://if.uy](https://if.uy)

## Prepare requirement
```
pip install pingparsing flask requests configparser
```
## Clone client
```
git clone https://github.com/tusik/ipcheck-client-python.git ipcheck
```

## Edit config file
```
cd ipcheck
vim conf
[server]
port=19970
debug=0
```
