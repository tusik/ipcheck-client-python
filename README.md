# Global Ping project
It's the client program for [https://if.uy](https://if.uy)

## Prepare requirement
```
pip3 install pingparsing flask requests configparser flask-cors
```
## Clone client
```
git clone https://github.com/tusik/ipcheck-client-python.git ipcheck
```

## Edit config file
```
cd ipcheck
vim conf
[server]
port = 19970
name = aaa
uuid =  
debug = 0
host = 127.0.0.1
```
